ScrollStory

A tiny Editor and Runtime for creating Scroll Stories.

The Vision

The simplest scroll-based animation engine. Elements have three states: in, on, and out. As you scroll, elements animate between these states.

How It Works

Each actor element has a natural "on stage" position defined by its CSS. The "in" state is before this position (faded and offset), and "out" is after (faded and offset the other way).

The Magic

Scroll to see elements animate. The runtime calculates when each element should start its in/out animations based on scroll position.

Get Started

This is just the beginning. The editor will let you visually configure these animations with a simple timeline interface.
