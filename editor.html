ScrollStory

Tiny Editor & Runtime

A tiny Editor and Runtime for creating Scroll Stories.

The Vision

The simplest scroll-based animation engine. Elements have three states: in, on, and out.

Box 1

How It Works

Each actor element has a natural "on stage" position. The "in" state is before this position, and "out" is after.

Box 2

The Magic

Scroll to see elements animate. The runtime calculates when each element should start its in/out animations.

Box 3

Get Started

This is just the beginning. The editor lets you visually configure these animations.
